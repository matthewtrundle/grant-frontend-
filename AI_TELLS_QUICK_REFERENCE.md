## Quick Copy Audit Test

Read any sentence and ask:
1. **Sounds like a press release?** ‚Üí Rewrite in plain English
2. **Could apply to any SaaS product?** ‚Üí Add specificity
3. **Would a human say this?** ‚Üí Read it aloud, rewrite if awkward
4. **Shows feature not benefit?** ‚Üí Add "so you can [outcome]"
5. **Unverifiable claim?** ‚Üí Delete or add proof link

## Anti-Pattern Checklist (Use Before Publishing Anything)

### ‚ùå DELETE THESE IMMEDIATELY
- [ ] "Journey" (unless literal travel)
- [ ] "Leverage" (use "use" instead)
- [ ] "Empower" (say what you actually do)
- [ ] "Comprehensive" (be specific instead)
- [ ] "Robust" (means nothing)
- [ ] "Cutting-edge" (says who?)
- [ ] "Industry-leading" (prove it)
- [ ] Unverified statistics
- [ ] Generic testimonials
- [ ] Stock photos

### ‚úÖ ADD THESE EVERYWHERE
- [ ] Specific numbers: "$2.4M" not "$2M+"
- [ ] Real names with photos/videos
- [ ] Proof links (LinkedIn, grant databases)
- [ ] Founder voice ("I" or "we")
- [ ] Plain English (8th grade level)
- [ ] Benefits not features
- [ ] Personality/humor
- [ ] Asymmetry (break the grid)
- [ ] Confetti/celebration
- [ ] Urgency (real deadlines)
